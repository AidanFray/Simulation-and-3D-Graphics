# Simulation-and-3D-Graphics
A university graphics and simulation project.

# Controls
     MOVEMENT (If Free Moving camera)
     a - Move Left
     d - Move Right
     
     w - Move Forward
     s - Move Backwards

     k - Rotate Clockwise
     l - Rotate Anti-Clockwise

     r - Y axis up
     f - Y axis down

     CAMERA
     v - Static Camera
     b - Fixed Path(Reset)
     n - Free Moving
     m - Follow Sphere

     TOGGLES
     g - Toggle Gravity
     p - Toggle CrazyMode
     h - Toggle Intergeneration method
     1 - Toggle Fullscreen mode
     0 - Toggle view mode

     SPHERES
     z - Add new Sphere
     c - Clear all Spheres
